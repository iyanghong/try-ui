# <center>Try Ui</center>
> A Vue.js 2.0 UI Toolkit for Web.

## Install
``` shell 
npm install try-ui
```

## Quick Start
```javascript
import Vue from 'vue'
import TryUi from 'try-ui'

Vue.use(TryUi)

// or
import {
    Input,
    Button
    // ...
} from 'try-ui'

Vue.component(Input.name, Input)
Vue.component(Button.name, Button)
```

## Browser Support
Modern browsers and Internet Explorer 10+.
